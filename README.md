## 新鲜出炉 (2021-01)

### 2021-01-29 \[问题\]

**问题描述**

本地 Git 提交时，出现问题 **Logon failed, use ctrl+c to cancel basic credential prompt to Git**。

**解决方案**

将 Git 升级到最新版本。

**具体做法**

Windows：`git update-git-for-windows`

Linux：`git update`

或者使用以下链接直接下载最新版本的 binary 重新安装：

https://git-scm.com/downloads
